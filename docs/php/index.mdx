<CardChapter>

  ### Fehler abfangen

Es kommt zu einer Fehlermeldung, wenn PHP-Code <b><u>und</u></b> das Formular in einer Datei vorliegen.

PHP will Werte aus dem Formular direkt nach dem Laden der Seite verarbeiten,diese liegen aber nicht vor, 
da das Formular nicht abgesendet wurde.

```cpp 
  <form action="#" method="get">
    <input type="number" name="zahl">
    <input type="submit">
  </form>

  <?php

// highlight-next-line-error
  echo ($_GET["zahl"]);

  ?>
```
  
Dies kann durch `isset()` abgefangen werden, da der Code nur ausgeführt wird, wenn 
das Formular abgesendet wurde und die Werte der Eingabefelder vorliegen.

```php 
  <form action="#" method="get">
    <input type="number" name="zahl">
    <input type="submit">
  </form>
  <?php

// highlight-next-line
  if (isset($_GET["zahl"])) {

  echo ($_GET["zahl"]);
  // highlight-next-line
  }

  ?>
```

</CardChapter>
<br/>

<CardChapter>

  ### Daten senden

- `action="#"` PHP-Dokument für die Bearbeitung befindet sich im aktuellen Dokument
```php 
  <form action="#" method="get">
```

</CardChapter>
<br/>

<CardChapter>

  #### `GET`

- Bei Methode `method="get"` sind die übergebenen Werte in der Browser-URL einsehbar
```php 
  <form action="#" method="get">
```

</CardChapter>
<br/>

<CardChapter>

  #### `POST`

- Bei Methode `method="post"` sind die übergebenen Werte `nicht` in der Browser-URL einsehbar
- Passwortschutz ist gewährleistet!
```php 
  <form action="#" method="post">
```

</CardChapter>
<br/>

<CardChapter>

  ### Daten verarbeiten

Je nach Übergabemethode können die Werte der Formulare an PHP zwecks Weiterverarbeitung übergeben werden.
Über den Namen des Eingabefeldes kann dies erfolgen `name="zahl"` 
- `method="get"`
  
  ```php 
  // highlight-next-line
  <form action="#" method="get">
  // highlight-next-line
    <input type="number" name="zahl">
    <input type="submit">
  </form>

  <?php
  if (isset($_GET["zahl"])) {

    // highlight-next-line
    $betrag = $_GET["zahl"];
    echo $betrag;
  }
  ?>
  ```
  
- `method="post"`
  ```php
  // highlight-next-line
    <form action="#" method="post">
    // highlight-next-line
      <input type="number" name="zahl">
      <input type="submit">
    </form>

    <?php
    if (isset($_POST["zahl"])) {

  // highlight-next-line
      $betrag = $_POST["zahl"];
      echo $betrag;
    }
    ?>
  ```

<CardChapter>

  ##### `sort()` `rsort()`

Sortiert ein Array nach den Werten. Zuordnung von Schlüssel und Wert bleibt nicht erhalten, 

<CardDanger>

Danger:
- Ein assoziatives Array wird in numerisch indiziertes Array umgewandelt.

</CardDanger>

- `sort()` 
  - sortiert aufsteigend 
- `rsort()`
  - sortiert absteigend

```php 
$var = array("c", "b", "a");
sort($var);
print_r($var);
```

```php 
$var = array("a", "b", "c");
rsort($var);
print_r($var);
```

</CardChapter>

  #### Array & Variablen

<CardChapter>

  #### `empty()`

- liefert `true`, wenn Variable nicht definiert, leer oder 0 ist.

```php 
$var = 10;

if (empty($var)) {
  echo "empty";
} else {
  echo "not empty";
}
```

```php 
$var = array(rand(0, 10));

if (empty($var)) {
  echo "empty";
} else {
  echo "not empty";
}
```

</CardChapter>
<br/>


<CardChapter>

  #### `isset()`

- Prüfung, ob Variable oder Array vorhanden
- liefert `true`/`false`

```php 
if (isset($_GET["zahl"])) {

  echo ($_GET["zahl"]);
}
```

</CardChapter>
<br/>

<CardChapter>

  #### `unset()`

- Variable oder Array löschen / zurücksetzen

```php 
$var = 10;

echo $var;
unset($var);
// highlight-next-line-error
echo $var;
```

```php 
$arr = array(rand(0, 10));

print_r($arr);
unset($arr);
// highlight-next-line-error
print_r($arr);
```

</CardChapter>
<br/>

<CardChapter>

  #### `print_r()`
- Ausgabe von Werten
  - Variablen
    ```php 
    $var1 = 11;

    print_r($var1);
    ```
  - Array
    ```php 
      $arr1 = array(rand(0, 10));

      print_r($arr1);
    ```

</CardChapter>
<br/>

<CardChapter>

  #### `var_dump()`

- Ausgabe Datentyp und Wert

  ```php 
  var_dump($var1);
  var_dump($var2, $var3);
  ```

</CardChapter>
