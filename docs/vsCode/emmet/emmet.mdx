  ### enable tab completion

  ```json title='settings.json'
  "emmet.triggerExpansionOnTab": true,
  ```  
</CardChapter>
<br/>



<CardChapter>

  ### include / exclude languages

  ```json title='settings.json'
  "emmet.excludeLanguages": [],
    "emmet.includeLanguages": {
    "typescript": "typescriptreact",
    "javascript": "javascriptreact",
    "vue-html": "html",
    "astro": "html",
  },
  ```  
</CardChapter>
<br/>



<CardChapter>

  ###  set language

  ```json title='settings.json'
  "emmet.variables": {
      "lang": "de"
  },
  ```
