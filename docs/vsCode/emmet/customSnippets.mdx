  ### Comment Anchor

  - PATH
    ```json title='p.code'
    "commentAnchorPath": {
      "prefix": "cap",
      "body": [
        "$BLOCK_COMMENT_START PATH ${1:Text:} $BLOCK_COMMENT_END",
        "$2",
      ],
      "description": "Anchor PATH"
    },
    ```

- Note
    ```json title='p.code'
    "commentAnchorNote": {
      "prefix": "can",
      "body": [
        "$BLOCK_COMMENT_START NOTE ${1:Text:} $BLOCK_COMMENT_END",
        "$2",
      ],
      "description": "Anchor PATH"
    },
    ```

- Section
    ```json title='p.code'
    "commentAnchorSection": {
      "prefix": "cas",
      "body": [
        "$BLOCK_COMMENT_START SECTION ${1:Text:} $BLOCK_COMMENT_END",
        "$2",
        "$BLOCK_COMMENT_START !SECTION END ${1:Text:} $BLOCK_COMMENT_END"
      ],
      "description": "Comment Anchor Section"
    },
    ```

</CardChapter>

<CardChapter>

  #### Highlight

- Higlight Line
    ```json title='p.code'
	"HighlightCommentCodeblock": {
		"prefix": "//n",
		"body": [
			"// highlight-next-line"
		],
		"description": "Highlight next CodeLine"
	},
  ```

- Highlight Error
  ```json title='p.code'
  "HighlightErrorCodeblock": {
    "prefix": "//err",
    "body": [
      "// highlight-next-line-error"
    ],
    "description": "Highlight Error next CodeLine"
  },
  ```

  ```json title='p.code'
  // highlight-next-line-error  
  ```


- Higlight Block
  ```json title='p.code'
  "HighlightBlockCodeblock": {
    "prefix": "//block",
    "body": [
      "// highlight-start",
      "// highlight-end"
    ],
    "description": "Highlight Codeblock"
  },
  ```

</CardChapter>
