  ## Git

  - Add Remote
  
    `Strg` `Shift` `P` "Add Remote"
  - Undo last commit
  
    `Strg` `Shift` `P` "Undo Last commit"
  - Remove Remote
  
    `Strg` `Shift` `P` "Remove Remote"
  
  - Ignoring Files
    ```.gitignore title='.gitignore'
    # ide
    .vscode/

    # Dependencies
    /node_modules

    # Production
    /build
    .env

    # Generated files
    .docusaurus
    .cache-loader

    # Misc
    .DS_Store
    .env.local
    .env.development.local
    .env.test.local
    .env.production.local

    npm-debug.log*
    yarn-debug.log*
    yarn-error.log*
    ```
